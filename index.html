How does ibeacon work ?
IT-specialist implements works with the restaurant’s server that is responsible for the work of the software R-Keeper, sets xml-interfaces for data exchange, sets the printing parameters to receipts etc.
The restaurant receives an account in the system. Web-interface allows to the manager to select and edit the parameters of courses, special campaigns, to review and analyze the statistics of orders. The complete restaurant menu is received by the system from the database of R-Keeper, with the help of set interfaces.
IT-specialists set and adjust the beacons of iBeacon system or other BLE-devices (these devices allow to follow the location of clients’ smartphones). It is expected that one beacon is to be set per every table included into the system
IT-specialists sort coordinates or fingerprints between the physical tables in the restaurant. The received data is editable via the web-interface.
